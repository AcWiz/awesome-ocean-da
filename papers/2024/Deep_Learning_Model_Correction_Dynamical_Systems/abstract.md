# Deep Learning for Model Correction of Dynamical Systems with Data Scarcity

## 基本信息
- **arXiv**: [2410.17913](https://arxiv.org/abs/2410.17913)
- **作者**: Caroline Tatsuoka, Dongbin Xiu
- **年份**: 2024

## 中文总结
**核心贡献**：提出一种仅需稀缺高保真数据集即可修正现有动力系统模型的深度学习框架。

**主要方法**：
- 首先使用深度神经网络(DNN)近似低保真模型
- 通过迁移学习(TL)利用稀缺高保真数据进行模型修正
- 不对修正项的具体形式做任何假设

**意义**：解决了数据稀缺情况下无法进行数据驱动建模的难题，为海洋动力系统等数据获取成本高的应用场景提供了新的修正策略。
