# Using Deep Learning to Identify Initial Error Sensitivity for Interpretable ENSO Forecasts

## 基本信息
- **arXiv**: [2404.15419](https://arxiv.org/abs/2404.15419)
- **作者**: Kinya Toride, Matthew Newman, Andrew Hoell, Antonietta Capotondi, Jakob Schlör, Dillon J. Amaya
- **年份**: 2024

## 中文总结
**核心贡献**：提出可解释性深度学习方法"优化模型类比"(optimized model-analog)，用于识别ENSO预报中初始误差敏感区域。

**主要方法**：
- 采用卷积神经网络(CNN)估计状态依赖权重
- 结合模型类比预报方法识别初始类比状态
- 追踪物理驱动的系统演化过程

**意义**：揭示了厄尔尼诺和拉尼娜预报对不同初始状态的敏感性差异，为季节到年际尺度的海洋-大气预报提供可解释性框架。在热带太平洋SST预报中实现了10%的提升。
