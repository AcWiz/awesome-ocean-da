# 投稿指南

欢迎提交新的论文到本项目！

## 投稿要求

1. **论文范围**：AI/深度学习与数据同化相关的研究
2. **文件格式**：
   - PDF 文件放入对应年份目录
   - 每篇论文需包含 `abstract.md` 总结文件

## 文件命名规范

```
papers/[年份]/[论文简称]/
├── abstract.md    # 必须：论文摘要和总结
└── paper.pdf      # 可选：论文 PDF
```

## abstract.md 模板

```markdown
# [论文标题]

## 基本信息

- **arXiv/链接**: [链接地址]
- **作者**: 作者列表
- **年份**: 年份

## 中文总结

**核心贡献**：简要描述论文的主要贡献

**主要方法**：论文采用的主要方法和技术

**意义**：该研究的意义和影响

## 关键词

- 关键词1, 关键词2, ...
```

## 更新 README

提交 PR 时，请同时更新根目录的 `README.md` 论文列表。
